# Лендинг эфира — «Как воспитать ребёнка мусульманином»

Одностраничный лендинг для бесплатного эфира Солихи умм Муса. Весь сайт построен как
единый «маршрут»: золотая дорога вьётся через все секции, а маркер «вы здесь» движется
по ней при прокрутке.

## Как опубликовать на GitHub Pages

1. Создайте репозиторий на GitHub и загрузите в него **всё содержимое этой папки**
   (включая скрытый файл `.nojekyll` и папку `_ds`).
2. В репозитории откройте **Settings → Pages**.
3. В разделе **Build and deployment** выберите источник **Deploy from a branch**,
   ветку `main` и папку `/ (root)`. Сохраните.
4. Через 1–2 минуты сайт будет доступен по адресу
   `https://<ваш-логин>.github.io/<имя-репозитория>/`.

> ⚠️ Файл **`.nojekyll` обязателен**: без него GitHub Pages игнорирует папку `_ds`
> (имена, начинающиеся с `_`), и сломаются стили и шрифты.

## Что нужно настроить перед публикацией

- **Ссылка на Telegram-бот.** Откройте `index.html`, найдите строку
  `TG_BOT_URL_PLACEHOLDER = "[ВСТАВИТЬ_ССЫЛКУ_НА_ТГБОТ]"` и замените текст в кавычках
  на ссылку вашего бота (например `https://t.me/your_bot`). Это адрес, куда ведёт
  кнопка «Иду на эфир».

## Структура

```
index.html              — страница (откройте/задеплойте её)
support.js              — рантайм компонента
.nojekyll              — отключает Jekyll на GitHub Pages (не удалять)
_ds/                    — дизайн-система (токены, шрифты, компоненты)
assets/
  hero.mp4             — фоновое видео в hero
  hero.png             — постер hero (пока грузится видео)
  danger-smoke.mp4     — видео дыма в секции «Опасный участок»
  danger-phone.png     — постер этого видео
  author-avatar.png    — аватар спикера
  backdrop.png         — фон секции «Куда вы приедете»
```

## Примечания

- Шрифты (Playfair Display, Montserrat, Allura) подгружаются из Google Fonts —
  нужен доступ в интернет.
- Видео проигрываются без звука, зациклены; видео дыма грузится лениво (только при
  подходе к секции), поэтому начальная загрузка остаётся быстрой.
- Страница адаптивна: на телефоне дорога-маршрут упрощается, секции складываются
  в одну колонку.
